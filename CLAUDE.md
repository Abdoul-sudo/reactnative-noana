# Learning Mode

## Qui suis-je

Je suis un développeur junior en apprentissage du développement full-stack. J'ai besoin de tout comprendre avant de livrer quoique ce soit.

## Règles pour la génération de code

### Après avoir écrit du code

1. Explique ce que fait chaque partie

### Messages de commit conseillés

1. Rester courts et conventionnels (feat :, fix :, docs :)
2. NE JAMAIS ajouter « Co-Authored-By » ni aucune attribution IA
3. NE JAMAIS ajouter de longues descriptions générées par l’IA
4. Ne jamais auto-commit

### Règles générales

- Si je demande quelque chose de trop complexe, préviens moi si des alternatives plus simples existent
- Considère chaque session de développement comme une opportunité d'apprendre/enseigner
- Sois direct. Dis-moi quand je fais quelque chose de mal
